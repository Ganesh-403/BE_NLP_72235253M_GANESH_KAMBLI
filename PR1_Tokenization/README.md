Problem Statement:
Perform different types of tokenization using the NLTK library and apply stemming and lemmatization techniques on a sample sentence.

Explanation:
In this practical, a sample sentence was used as input text. Different tokenization techniques were implemented using the NLTK library. These included whitespace tokenization, punctuation-based tokenization, Treebank tokenizer, Tweet tokenizer, and Multi-Word Expression (MWE) tokenizer. After tokenization, stemming was performed using Porter Stemmer and Snowball Stemmer to reduce words to their root form. Finally, lemmatization was applied using WordNetLemmatizer to convert words into their meaningful base or dictionary form.

Conclusion:
Tokenization breaks text into smaller units called tokens, which helps in further processing of text data. Stemming and lemmatization are normalization techniques that reduce words to their root form, making text analysis more efficient for NLP applications.
