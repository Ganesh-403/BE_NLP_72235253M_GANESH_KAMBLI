Problem Statement:
Perform tokenization, stemming and lemmatization using NLTK.

Explanation:
Different tokenization techniques such as whitespace, punctuation,
treebank and tweet tokenizer were implemented. Porter and Snowball
stemmer were used for stemming. WordNet lemmatizer was used for
lemmatization.

Conclusion:
Tokenization and normalization techniques help prepare text data
for NLP tasks by converting raw text into structured tokens.
