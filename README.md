BE NLP Laboratory Practicals

This repository contains the implementation of Natural Language Processing (NLP) laboratory practicals for Laboratory Practice VI (LP-VI) in the Final Year Computer Engineering (SPPU 2019 Pattern).

The practicals demonstrate fundamental NLP preprocessing techniques and text representation methods using Python and popular NLP libraries.

Technologies Used

Python

NLTK (Natural Language Toolkit)

Scikit-learn

Gensim

Pandas

These libraries are used for performing text preprocessing, tokenization, vectorization, and word embeddings.
